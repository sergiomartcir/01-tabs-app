# Proyecto Tabs con Ionic 8 - Angular 20

## Instrucciones para ejecutar

### 1. Servidor API (Node.js)
1. Ve a la carpeta `events-api`.
2. Ejecuta `npm install` para instalar dependencias.
3. Ejecuta `node index.js` para arrancar el servidor (http://localhost:3000).

### 2. Aplicación (Ionic)
1. Ve a la carpeta `ionic-app`.
2. Ejecuta `npm install`.
3. Ejecuta `ionic serve` para ver la aplicación en el navegador.
